Name
A name for this scheduled task.

Description
A brief description of what this task does and why.

URL or script path
If a URL, then this URL will be called and considered successful if the return status code is "200"

If a script path, then the normal Myna Path translations apply. The script will be executed in a standalone thread. Scripts stored in /shared/js/cron or /WEB-INF/ are not accessible via URL which makes those directories good choices for sensitive cron scripts.

Active?
Only active tasks will be executed.

Start Date
The first date this script should be run.

Start Time
The first time this script will run in hours(24):minutes:seconds format

End Date
This is the last date the URL or script should run. All normal times the script would run are executed but no further executions will be preformed after this date
