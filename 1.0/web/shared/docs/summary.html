Summary:
Myna is Java web application (.war file) that provides dynamic web content from both interpreted and compiled JavaScript, as well as complete access to Java libraries.
Goals:
To provide an open source rapid web development platform that:
  • Can be distributed as a simple ".war" file.
  • Can leverage Java's large number of libraries.
  • Can be easily administered online.
  • Provides a rich standard library to speed development.
  • Provides easy access to databases.
  • Can run on JVM 1.5+
  • Can be deployed on top of any Java Servlets container
  • Can be distributed freely. All distributed code covered under Open Source Licenses.
... but does NOT:
  • Provide an MVC framework. Developers are free to implement their own framework if appropriate.
  • Require Java programing to extend. As much behavior as practical will be implemented in JavaScript.
