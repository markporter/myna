$lib. appendFunction

Appends supplied function to the event chain.

Parameters

objobject to modify
functionNamename of the function on the object to modify
functionObjfunction object to append

Returns

reference to the new function

Detail

Existing functions are preseved and executed in the order they were declared.  The resulting chain will always return a reference to itself, regardless of the original return value of the function.

Examples

$lib.appendFunction($session,”onSessionStart”,function(){$res.print(‘session started’)}); $lib.appendFunction(window,”onload”,function(){alert(“window loaded!”)});

$lib. checkObjectRequired

Ensures an object has certain properties defined.

Parameters

objobject to check
requiredArray of proprty name strings to look for

Returns

void

Detail

This function is intended for Javascript Objects being used as data containers.  Particularly JS objects passed as function parameters.

This function simply checks to see if every string in the “required” array has a corrosponding property in “obj”.  The first time a property is not found an exception is raised
