appendFunction

appendFunction:function(obj,
functionName,
functionObj)

Appends supplied function to the event chain.

Parameters

objobject to modify
functionNamename of the function on the object to modify
functionObjfunction object to append

Returns

reference to the new function

Detail

Existing functions are preseved and executed in the order they were declared.  The resulting chain will always return a reference to itself, regardless of the original return value of the function.

Examples

$lib.appendFunction($session,"onSessionStart",function(){$res.print('session started')});
$lib.appendFunction(window,"onload",function(){alert("window loaded!")});

checkObjectRequired

checkObjectRequired:function (obj,
required)

Ensures an object has certain properties defined.

Parameters

objobject to check
requiredArray of property name strings to look for

Returns

void

Detail

This function is intended for Javascript Objects being used as data containers.  Particularly JS objects passed as function parameters.

This function simply checks to see if every string in the required array has a corresponding property in obj.  The first time a property is not found, an exception is raised.

importLib

importLib:function(object,
includeSubObjects)

Copies the functions of the supplied object to the global scope.

Parameters

objectObject to import
includeSubObjectsOptional, default false Should sub-objects be imported as well?

Returns

void

Detail

This is primarily intended for importing functions from the $lib object.  This code...

$lib.importLib($lib,true)

... will import the entire standard library to the global scope allowing the use of

var version = readFileStirng(file);

in place of

var version = $lib.file.readFileStirng(file);

It is recommended that code that is intended to be shared NOT import any libraries, to reduce the chance of name collision.

setObjectDefaults

setObjectDefaults:function (obj,
defaults)

sets default properties on supplied object

Parameters

objObject to modify
defaultsObject that represents the default properties

Returns

void

Detail

Every property in defaults is checked against obj.  If the property is undefined in obj, it is copied from defaults.

Example

$lib.setObjectDefaults($res.data, {
    name:"bob",
    isDeceased:false
});

typeOf

typeOf:function(object)

an enhanced replacement of the the Javscript builtin typeof function.

Parameters

objectobject to inspect

Returns

a string representing the type of the object

Detail

The builtin JavaScript typeof function does not identify some stamndard objects, specifically Arrays, Dates, and Nulls.  When running in Myna, it is also important to know when the object is a Java object.  This function returns the standard typeof strings as well ass the following:

  • null
  • array
  • class
  • date
