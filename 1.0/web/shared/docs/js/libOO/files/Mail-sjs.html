Myna.Mail = function (options)

Construct a Mail object

Parameters

optionsAn Options Object that contains configuration information for this mail object.  See below

Options

from”From” email address
toComma separated list of “to” email addresses
ccOptional default ‘’ Comma separated list of “cc” email addresses
bccOptional default ‘’ Comma separated list of “bcc” email addresses
subjectOptional default ‘’ email subject
bodyOptional default ‘’ email body.  If this is html, be sure to set isHtml to true
isHtmlOptional default false If true, body will be interpreted as html
attachmentsOptional default [] Optional array of Myna.File objects or MynaPath strings that represent files to attach to this email
