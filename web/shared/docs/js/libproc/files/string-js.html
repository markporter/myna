escapeHtml

$lib.string.escapeHtml=function(string)

replaces common symbols with their HTML entity equivalents

Parameters

stringString to convert

Returns

converted string

Detail

escapes the following symbols:

<   becomes &lt;
>   becomes &gt;
'   becomes &apos;
"   becomes &quot;

dateFormat

function dateFormat(date,
format)

Returns a formatted string version of the supplied date.

Parameters

dateDate object
formatString contining special formating characters.  See below

Format Characters

dDay of the month as digits; no leading zero for single-digit days.
ddDay of the month as digits; leading zero for single-digit days.
dddDay of the week as a three-letter abbreviation.
ddddDay of the week as its full name.
mMonth as digits; no leading zero for single-digit months.
mmMonth as digits; leading zero for single-digit months.
mmmMonth as a three-letter abbreviation.
mmmmMonth as its full name.
yyYear as last two digits; leading zero for years less than 10.
yyyyYear represented by four digits.
hhours; no leading zero for single-digit hours (12-hour clock)
hhhours; leading zero for single-digit hours (12-hour clock)
Hhours; no leading zero for single-digit hours (24-hour clock)
HHhours; leading zero for single-digit hours (24-hour clock)
Mminutes; no leading zero for single-digit minutes
MMminutes; a leading zero for single-digit minutes
sseconds; no leading zero for single-digit seconds
ssseconds; leading zero for single-digit seconds
lmilliseconds.
tone-character time marker string, such as A or P
ttmultiple-character time marker string, such as AM or PM

Example

<%=$lib.dateFormat(new Date(),"mm/dd/yyyy HH:MM:ss.l")%>

parseJson

$lib.string.parseJson=function(text)

Converts a JSON (http://www.json.org) string into an object

Parameters

textString to convert

Returns

Number String Array or Object contained in the JSON text

Throws

SyntaxError if not properly formatted

Detail

Adapted from http://www.json.org/json.js

This function expects strings to be properly formatted.  In particular watch out for property names, they must be quoted.

Bad:
{name:"bob"}

Good:
{"name":"bob"}

toJsonString

$lib.string.toJsonString=function(obj)

Converts the supplied object to JSON (http://www.json.org)

Parameters

objobject to convert

Returns

JSON string that represents obj

Detail

Adapted from http://www.json.org/json.js

Attempts to convert obj to JSON.  This is best used on simple Objects and Arrays.

Example

//this code might be called by an AJAX callback
var qry = $lib.sql.query("ds1","select * from users");
$res.print($lib.string.toJsonString(qry.data));
