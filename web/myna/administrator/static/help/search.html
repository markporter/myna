Logged At
Pick a date here to narrow the range. Either field can be blank. Dates are from midnight, inclusive.

Label
Filter on a log label. This can be multiple whitespace separated strings.

By default, each token is AND'd
Tokens starting with "!" are negative filters
Tokens containing a pipe "|" are OR'd

Example: 

     task post !next adult|ped
This will search for logs with a label containing task* and post* and (adult* or ped*) but NOT next*

Type
Filter on a log type. This can be multiple whitespace separated strings, like label searches. You can also click the underlined type name in the results to filter by that type.

App Name
Filter on an app name. This can be multiple whitespace separated strings, like label searches. You can also click the underlined app name in the results to filter by that app.

Request ID
Filter on a request ID. You normally click on the underlined request ID in the results to filter by this value.

Host Name
Filter on a host name. This can be multiple whitespace separated strings, like label searches. You can also click the underlined host name in the results to filter by that app.

Instance ID
Filter on an instance ID. This can be multiple whitespace separated strings, like label searches. You can also click the underlined instance ID in the results to filter by that instance.

Purpose
Filter by instance purpose. This can be multiple whitespace separated strings, like label searches. You can also click the underlined purpose in the results to filter.
